# Opérations fonctionnelles
 - Lecture du fichier de test (.c)
 - Analyser lexical
 - Analiser syntaxique
 - Analiser sémantique
 - Génération de code :
    - opérations arithmétiques
    - constantes
    - opérateurs unaires
    - blocs
    - expressions
    - conditions (if, else)
    - comparaison ( =, !=, <, >, <=, >=)
 - Boucles :
    - While

# Utilisation
Pour lancer l'application, ouvrir un invite de commande et lancer : 
Compil.exe [nomFichierC]
Exemple :
Compil.exe c:\test.c
